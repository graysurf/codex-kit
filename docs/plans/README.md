# Plans

Implementation plan documents generated by planning skills (e.g. `create-plan`, `create-plan-rigorous`) live here.

Naming convention: `docs/plans/<kebab-case>-plan.md`.

## Format

- Plan format: `docs/plans/FORMAT.md`
- Tooling and commands: `docs/plans/TOOLCHAIN.md`
- Workflow runbook: `docs/runbooks/plan-workflow.md`
