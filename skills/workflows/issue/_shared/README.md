# skills/workflows/issue/\_shared

Purpose: shared assets for issue-related skills under `skills/workflows/issue/`.

Allowed contents:

- `lib/` (shared issue helpers)
- `references/` (issue docs/specs)
- `assets/` (templates, fixtures)
- `python/` (language-specific helpers)

Forbidden:

- `SKILL.md`
- `scripts/`
- `tests/`

Keep entrypoints in per-skill `scripts/` directories.
