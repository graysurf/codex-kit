# skills/workflows/plan/_shared

Purpose: shared assets for plan-related skills under `skills/workflows/plan/`.

Allowed contents:

- `lib/` (shared plan helpers)
- `references/` (plan docs/specs)
- `assets/` (templates, fixtures)
- `python/` (language-specific helpers)

Forbidden:

- `SKILL.md`
- `scripts/`
- `tests/`

Keep entrypoints in per-skill `scripts/` directories.
